How to read this dashboard — green = hit, yellow = close, red = missed

What you're looking at: Every row is a daily habit. Every column is a day. Green = completed, yellow = close to target, red = missed, gray = no data yet.

Why 23 habits: As CEO of a longevity company, Lucas tracks across every domain of the Daybreaker protocol: sleep, nutrition, movement, prescriptions, aesthetics, therapies, environment, and performance. The breadth is intentional.

How data gets here: The Streaks iOS app syncs via iCloud. A script extracts the backup, generates the dashboard data, and deploys to this page. No manual editing is possible. The "last synced" timestamp above is proof of freshness.

Numeric habits (sleep hours, protein grams, water oz, calories) show the actual measured value in each cell, not just pass/fail.
